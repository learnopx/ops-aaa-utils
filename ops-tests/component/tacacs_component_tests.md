## Test addition of TACACS+ server (with no optional parameters)
### Setup
#### Topology diagram
```ditaa

```
### Description
- Add a TACACS+ server using just the IPV4 address.
- Add a TACACS+ server using just the FQDN.

#### Test pass criteria
